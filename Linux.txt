Terminal:
7z a test.7z . [file-names] - create 7zip-archive
tar -xvf filename.tar.gz ~/Desctop - unarchive data from archive to Desctop
history - check history of commands (CTRL + R - find command)
ln -s ~/Programs/<folder>/bin/<file> /usr/local/bin - add program(file) in PATH (write command and run program) | ngrok
nc/ncat -lvp 4444 > file_name.txt - to get file from other PC (nc - linux, ncat - windows)
nc/ncat -w 5 <ip-to> 4444 < file_name.txt - send file to other PC (nc - linux, ncat - windows)

Hotkeys:
CTRL + ALT + T - create new terminal
ALT + F4 - close program
ALT + F3 / Windows - find program
CTRL + L - clear (terminal)
CTRL + SHIFT + + - zoom in (terminal)
CTRL + - - zoom out (terminal)
CTRL + A - cursor in start string (terminal)
CTRL + E - cursor in end string (terminal)
CTRL + B - cursor for 1 symbol back/left (terminal)
CTRL + F - cursor for 1 sybmol next/right (terminal)
SHIFT + up/down - scroll up/down (terminal)
CTRL + R - find command in history (terminal)
CTRL + K - kill/cut (terminal)
CTRL + Y - paste (terminal)
CTRL + Tab - next workspace
CTRL + Space - press workspace
