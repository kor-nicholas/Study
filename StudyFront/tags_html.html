Строительные теги(блочные теги)

Блочные теги - каждый тег пишется с новой строки

Строчные теги - пишутся в 1 строку и зависят от количества контента(могут переходить на новую строку)

Правило №1 - нельзя помещать в строчный тег, блочный тег(наоборот - можно)

footer - предназначен для подвала сайта(так же можно помещать div теги)

form - тег всех форм (взаимодействие с пользователем, получение от его информации). Обязательный атрибут action - куда уйдут данные, атрибут method - http-метод запроса (get - пользователь видит параметры в адресной строке, post - передает данные скрытно), атрибут enctype = "multipart/form-data" - для передачи файлов

атрибут placeholder(для тега input и textarea) - подпись поля

атрибут disabled(все теги в форме, кроме самой form) - данные не отправляются + поле становится тусклым

атрибут tabindex(все теги в форме, кроме самой form) - порядок перехода между элементами в форме, при нажатии Tab

атрибут name - название параметра при передаче на сервер

атрибут value - значение параметра

address - для отображения контактной информации(например, в footer почта, телефон, адрес)

article - чаще всего используется для самой статьи

aside - бокова панель, сноска, метка

Строчные теги

span - выделенный текст

some text ... span text, that we need highilght yellow ... some text

a - ссылка(атрибут href - куда перейдет, при нажатии на ссылку, например на другую html-страницу, либо сайт, атрибут target="_blank" - при нажатии, содержимое откроется в новом окне браузера) ; в содержимом тега - text, img, другие строчные теги

img - вставляет изображение на страницу, не закрывающий тег (атрибуты src и alt : обязательны, src - путь, alt - текст, если картинка не загрузится)

input(в теге form) - принимает от пользователя информацию (атрибут type - определяет метод приема данных от пользователя, text - появится поле ввода текста, tel - для открытия цифровой клавиатуры на телефоне, password - при вводе, символы заменяются на звездочки, radio - поле с точкой выбора разных вариантов и отделяются именем-name, checked type = 'radio' - выбрать значение по-умолчанию, checkbox - флаг который пользователь может либо выбрать либо не выбрать и так же с помощью атрибута checked можно выбрать по-умолчанию, file - выбрать файл без value и текста в самом теге, атрибут placeholder - подпись поля), атрибут name - обязательно, что мы принимаем, value - если ввести, то данные появятся в поле ввода(например текста) и чтобы их изменить, человеку нужно будет сначала их удалить

textarea - поле ввода большого текста(атрибут name, атрибуты cols и rows - размеры поля, но можно не указывать, так как потом оно будет контролироваться с помощью css/js, атрибут placeholder - подпись поля, но оно не отправляется)

select - выпадающий список(коробка), атрибут name(multiple name = "" - чтобы можно было зажать Ctrl и выбрать несколько вариантов)

option - варианты выбора в выпадающем списке, пишется внутри тега select (атрибут value, текст внутри тега для отображения пользователю), selected value - выбрать по-умолчанию

button - кнопка(текст внутри тега - текст на копке, атрибут type = 'submit' - вся форма отправляется, type = 'reset' - вся форма очищается)

abbr - при наведении мышки к тексту, подсвечивается текст из атрибута title

text ... abbr text ... continue text

mark - подсвечивание/выделение/highilght выделенного в тег текста

text ... mark text ... continue text

ins - внизу подчеркнутый текст

text ... ins text ... continue text

sub - отобразить текст в виде нижнего индекса

text ... H2O ... continue text

sup - отобразить текст в виде верхнего индекса

text ... H2O ... continue text
